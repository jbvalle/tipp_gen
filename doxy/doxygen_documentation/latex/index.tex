This Project aims to demonstrate the usage of command line input arguments.\textbackslash{}

Tipp Generator simulates a Tipp Generator with X\+Xoutof\+YY tipps.\textbackslash{} These Tipps are generated randomly, individually unique and display in an increasing order.\textbackslash{}\hypertarget{index_autotoc_md1}{}\doxysection{Installation}\label{index_autotoc_md1}
Binaries are generated using make. Execute follow command in tippgen/src directory\textbackslash{}


\begin{DoxyCode}{0}
\DoxyCodeLine{make}
\end{DoxyCode}
\hypertarget{index_autotoc_md2}{}\doxysection{Usage}\label{index_autotoc_md2}
The program is executed by typing the following command in tippgen/src \textbackslash{}


\begin{DoxyCode}{0}
\DoxyCodeLine{./tippgen totaltipps XXoutofYY}
\end{DoxyCode}
